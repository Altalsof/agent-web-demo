Esta página sirve para probar tu canal Web Messaging / Agentforce en un dominio HTTPS.

Instrucciones:
  1. En Salesforce: Setup → Messaging → Deployments (o el Canal Web) y copia el Deployment Snippet.
  2. Pégalo justo antes de </body> en esta página.
  3. Asegúrate de agregar este dominio a la lista de dominios permitidos del deployment (Allowed Domains/Origins).

Si todo está correcto, verás el widget de chat 💬 en la esquina (según tu configuración de branding).
